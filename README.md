# 📊 Análisis de Datos – Alura Store

## 📌 Descripción del Proyecto

Este proyecto forma parte del **Challenge de Data Science de Alura LATAM**, cuyo objetivo es analizar los datos de ventas de cuatro tiendas de la cadena **Alura Store** para ayudar al Sr. Juan a tomar una decisión estratégica sobre cuál tienda debería vender para iniciar un nuevo emprendimiento.

Para ello, se realizó un análisis exploratorio de datos utilizando **Python**, **Pandas** y **Matplotlib**, evaluando diferentes métricas clave relacionadas con el rendimiento de cada tienda.

---

## 🎯 Objetivo del Análisis

Analizar el desempeño de las cuatro tiendas mediante diferentes indicadores para identificar cuál presenta **menor rendimiento** y así recomendar cuál debería vender el Sr. Juan.

## 📊 Análisis Realizados

Durante el proyecto se realizaron los siguientes análisis:

### 1️⃣ Ingresos Totales por Tienda

Se calcularon los ingresos totales de cada tienda sumando el valor de los productos vendidos.

### 2️⃣ Ventas por Categoría de Producto

Se analizaron las categorías más vendidas en cada tienda para identificar patrones de consumo.

### 3️⃣ Calificación Promedio de Clientes

Se evaluó la satisfacción de los clientes mediante el promedio de las calificaciones otorgadas a los productos.

### 4️⃣ Productos Más y Menos Vendidos

Se identificaron los productos con mayor y menor volumen de ventas en cada tienda.

### 5️⃣ Costo Promedio de Envío

Se analizó el costo promedio de envío en cada tienda para comprender el impacto en la experiencia del cliente.

---

## 📈 Visualización de Datos

Para facilitar la interpretación de los resultados, se crearon diferentes gráficos utilizando **Matplotlib**, entre ellos:

* Gráficos de barras
* Comparaciones entre tiendas
* Visualizaciones de ventas por categoría

Estas visualizaciones permiten identificar patrones y tendencias dentro de los datos analizados.

---

## 📌 Conclusión

Después de analizar los datos de ventas, las calificaciones de los clientes, la distribución de productos vendidos y los costos de envío, se concluye que **la Tienda 4 presenta el menor rendimiento general dentro de la cadena Alura Store**.

Por esta razón, se recomienda que el Sr. Juan considere vender esta tienda y enfocar sus recursos en las tiendas que presentan mejores indicadores de desempeño.

Este proyecto demuestra cómo el **análisis de datos puede apoyar la toma de decisiones estratégicas basadas en evidencia**.
